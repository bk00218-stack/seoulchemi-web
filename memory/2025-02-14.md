## mobile-glass 프로젝트 (서울케미 웹앱)

### 문제: 텍스트 색상이 안 보임
- CSS 변수 (`var(--text-primary)` 등)가 인라인 스타일에서 제대로 작동 안 함
- 라벨, 제목 등 텍스트가 흰 배경에서 안 보이는 문제

### 해결책
- CSS 변수 대신 **직접 색상값** 사용:
  - `var(--text-primary)` → `#212529`
  - `var(--text-secondary)` → `#6c757d`
  - `var(--text-tertiary)` → `#868e96`
