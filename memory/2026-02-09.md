## 서울케미 웹앱 (mobile-glass) - 판매전표 도수표 구현

### 완료된 작업
Zeus와 함께 OlwsPro 레거시 시스템의 도수표(diopter grid)를 웹으로 재구현

#### 도수표 최종 구조
- **하나의 연속 표** (가운데 기준 좌우 분할)
- **왼쪽**: 근시(-Sph) 영역
- **오른쪽**: 원시(+Sph) 영역
- **가운데 열**: `-000+`, `-025+`, `-050+` 형식으로 SPH 표시
- **세로(행)**: SPH 값 (0.00 ~ 15.00)
- **가로(열)**: CYL 값 (0.00 ~ -4.00) 양쪽 동일
- **커서**: 방향키로 자유 이동, 숫자키로 수량 입력
- **색상**: 연한 회색 통일 (#e8e8e0), 현재 행 핑크색 하이라이트

#### 키보드 단축키
- F5: 품목(브랜드) 선택
- F6: 상품 선택  
- F2: 주문 전송
- ESC: 초기화/포커스 해제
- 방향키: 셀 이동 (←→ CYL, ↑↓ SPH)
- 숫자키: 수량 입력
- Del/Backspace: 삭제

#### 레이아웃
- 왼쪽 패널: 160px (가맹점/브랜드/상품 선택)
- 중앙: 도수표 (flex 1fr)
- 오른쪽 패널: 180px (주문 목록)
