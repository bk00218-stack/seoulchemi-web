# 2026-02-13 작업 기록

## mobile-glass 대규모 업데이트

### 보안 강화
- `src/middleware.ts` - API 인증 보호, Rate Limiting (일반 100/분, 로그인 5/분)
- `next.config.ts` - Security Headers (HSTS, X-Frame-Options, XSS Protection 등)
- `src/lib/security.ts` - 인증 유틸리티 함수
- JWT_SECRET 환경변수 설정 (.env)

### 새 기능 추가
- `/dashboard` - 매출/주문 차트 (Recharts 라이브러리)
- `/api-docs` - OpenAPI 스펙 기반 API 문서 UI
- `/api/dashboard` - 대시보드 통계 API
- `/api/docs` - OpenAPI JSON 엔드포인트
- `/api/errors` - 클라이언트 에러 로깅

### PWA 설정
- `public/manifest.json` - 앱 매니페스트
- `public/sw.js` - Service Worker (오프라인 지원)
- `/offline` - 오프라인 페이지
- `src/components/PWARegister.tsx` - SW 등록

### 알림 시스템
- `src/components/NotificationBell.tsx` - 실시간 알림 컴포넌트 (30초 폴링)
- Layout에 통합됨

### 에러 핸들링
- `src/components/ErrorBoundary.tsx` - 클라이언트 에러 바운더리
- `src/lib/logger.ts` - 서버 로깅 유틸리티

### 초기 데이터 시딩
- `scripts/seed-admin.ts` - 관리자 계정 생성
- `scripts/seed-data.ts` - 브랜드, 카테고리, 배송구역 등
- `scripts/backup-db.ts` - DB 백업 스크립트

### 모바일 반응형
- 햄버거 메뉴 추가
- 사이드바 → 슬라이드 메뉴
- 모바일에서 데스크톱 요소 숨김
